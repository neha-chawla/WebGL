Please use a browser that supports "canvas"

Extra Features
1) "Undo Shape" button to undo the last shape drawn
2) "Randomize Clear Colored Canvas" button to draw a new randomly colored canvas
3) "Randomize Shape" button to randomly select the next shape drawn
4) "Randomize Coor" button to randomly select the next color used
5) "Opacity" slider to change the transparency of the color selected
